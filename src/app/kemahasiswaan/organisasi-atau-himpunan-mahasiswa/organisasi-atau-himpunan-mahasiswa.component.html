Q : Ada berapa Himpunan/organisasi di Udinus?

Answer

BEM (Badan Eksekutif Mahasiswa) tingkat Fak. ada 5, BEM U ( Badan Eksekutif Mahasiswa Universitas) ada 1, MPM ( Majelis Permusyawaratan Mahasiswa ) ada 1, DPM (Dewan Perwakilan Mahasiswa) Tingkat Fak. ada 5, DPM U ( Dewan Perwakilan Mahasiswa Universitas ) ada 1, Untuk Himpunan terdapat di setiap jurusan yang ada di bawah Fakultas

Q : Apa Manfaat masuk ke organisasi/himpunan?

Answer

Mendapat pengalaman tentang kerja tim, Pengalaman dibidang sosialisasi, Pengalaman dibidang lobbying dan masih banyak lagi.
