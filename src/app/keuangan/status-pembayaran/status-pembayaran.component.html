1. Q : Dimana saya bisa melihat status pembayaran?
    2. A : Di Siadin Mahasiswa.

  2. Q : Apakah ada keringanan biaya uang gedung untuk jalur PMDK?
    3. A : Ada, sebesar 25%.

  3. Q : Apa syarat untuk membayar wisuda?
    4. A : Sudah masuk di data yudisium.

  4. Q : Bagaimana jika mahasiswa sudah membayar uang SPP, tetapi di dalam data tidak terdaftar?
    5. A : Konfirmasi ke BIRO Keuangan.

  5. Q : Bagaimana jika mahasiswa ingin mengetahui informasi pembayaran uang gedung?
    6. A : Menghubungi biro keuangan, atau akses admisi dinus.ac.id/pmbonline
