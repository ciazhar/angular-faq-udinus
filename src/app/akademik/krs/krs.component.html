1. Q : Apa yang dimaksud KRS?
    2. A : KRS adalah Kartu Rencana Studi. Didalam KRS ada banyak data dan informasi. Diantaranya adalah Data Diri, Matakuliah beserta jadwal dan atribut proses perkuliahan selama 1 semester, SKS, Modul, Status, Ruang kuliah. Pada KRS juga terdapat nama dosen wali.

  2. Q : Bagaimana Mendapatkan KRS?
    3. A : Syaratnya adalah : tercatat menjadi mahasiswa, telah menyelesaikan administrasi keuangan, mengisi sendiri (memilih matakuliah dan jadwal yang telah ditawarkan) melalui SiAdin. Selanjutnya WAJIB diverifikasi melalui dosen wali. Apabila belum diverifikasi oleh dosen wali, KRS tidak syah (KRS tidak dapat digunakan untuk mengambil modul, absensi tidak dapat diinput oleh TU, nilai UTS dan UAS tidak dapat diinput oleh dosen pengampu).

  3. Q : Bagaimana proses input KRS?
    4. A : Mahasiswa yang akan memilih dan mengambil matakuliah, WAJIB membaca pengumuman akademik, mengetahui jadwal inputnya, masuk ke SiAdin masing-masing dan melakukan akses menu input KRS. Mahasiswa wajib memilih status kuliah Pagi / Malam. Pilihan tersebut berdampak sajian matakuliah beserta jadwal sesuai dengan sesi pagi / malam. mahasiswa dapat memilih sesuai dengan selera dan aktifitas pribadi. hasil input anda akan otomatis update di data kesediaan kursi tiap kelompoknya. Jika sudah tekan tombol simpan. dan lanjutkan verifikasi. (KRS yang anda input sudah otomatis ada di Inbox nya dosen wali anda).
