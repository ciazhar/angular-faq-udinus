1. Q : Dimana Mahasiswa dapat melihat transkrip?
    2. A : Di SiAdin atau di TU.

  2. Q : Bagaimana Cara mengambil transkrip?
    3. A : Meminta surat permohonan ambil transkrip di TU, dan menyerahkan fotocopy ATM.

  3. Q : Dimana Saya dapat mengambil transkrip?
    4. A : Di TU Fakultas.

  4. Q : Apabila nilai mata kuliah Saya dari Dosen B akan tetapi di dalam transkrip tertulis C, mengapa bisa begitu?
    5. A : Menghubungi Dosen yang bersangkutan.

  5. Q : Bagaimana mekanisme menghapus nilai / mata kuliah pada transkrip?
    6. A : Hubungi kepala program studi untuk melakukan konsultasi terlebih dahulu.
