1. Q : Kemana saya harus mengurus masalah aktifasi siadin mahasiswa, jika seandainya terjadi masalah dengan siadin saya?
    2. A : ke bagian tata usaha Udinus.

  2. Q : Bagaimana Mekanisme Reset Password atau dan Aktifasi Siadin melalui TU?
    3. A : Pada saat TU akan melakukan reset password, TU berhak dan wajib menanyakan nama lengkap, nama orang tua, nomor telpon, HP, email, tanggal lahir yang bersangkutan dan tanggal lahir orang tua.

  3. Q : Kenapa Saya tidak berhasil login SiAdin, akan tetapi Saya suda memasukkan password dengan benar ?
    4. A : Beberapa kemungkinan yaitu format NIM salah, atau NIM tidak ditemukan, atau salah password, atau Status akademik sudah berubah ke status Lulus, atau anda salah memasukkan password anda, bila lupa password sebaiknya reset password anda ke TU.

  4. Q : Dimana Saya bisa mendapatkan informasi aktifasi SiAdin?
    5. A : BIAK, TU Fakultas masing-masing, PSI.

  5. Q : Apa yang harus Saya lakukan apabila Saya lupa password SiAdin?
    6. A : Anda bisa ke TU Fakultas, PSI atau email ke psi.udinus@gmail.com, dengan menyiapkan data diri dan data orang tua secara lengkap. karena syarat aktifasi adalah melakukan cek dan update data.

  6. Q : Bagaimana merubah status akademik dari Mangkir berganti menjadi Aktif?
    7. A : Silakan mengajukan permohonan aktif dengan membuat surat permohonan ditujukan ke Wakil Rektor I dan akan diteruskan ke Biro Akademik (BIAK). Dan tidak boleh lupa WAJIB menyelesaikan administratif keuangan terlebih dahulu.
