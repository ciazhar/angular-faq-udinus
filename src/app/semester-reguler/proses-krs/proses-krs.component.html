1. Q : Apa itu KRS?
    2. A : KRS merupakan kepanjangan dari Kartu Rencana Studi yang berisi kewajiban kuliah yang harus diikuti mahasiswa.

  2. Q : Kapan input KRS di mulai?
    3. A : Sebelum perkuliahan dimulai. Jadwal tepatnya akan diinformasikan sebelumnya melalui SiAdin dan papan pengumuman.

  3. Q : Pengambilan total SKS yang bisa diambil per semester berdasarkan apa?
    4. A : Indeks Prestasi Semester (IPS).

  4. Q : Penyebab kenapa tidak bisa input KRS?
    5. A : Belum menyelesaikan administrasi keuangan, atau masalah teknis seperti koneksi yang kurang baik.

  5. Q : Bagaimana jika kita kehabisan kelas pada waktu input KRS dan dimana kita harus bertanya?
    6. A : Ruang Data pada TU Fakultas.

  6. Q : Apa saja syarat untuk bisa melakukan input KRS?
    7. A : Menyelesaikan administrasi keuangan dan melakukan herregistrasi.

  7. Q : Dimana saya bisa melihat nama dosen wali saya?
    8. A : Di siadin mahasiswa.
